## Objective
This project is a mini online banking system developed as part of an assignment.  
It demonstrates account creation, transaction handling, validation logic, and a clean, responsive UI.

The application is built using **HTML, CSS, and JavaScript** and is deployed as a public web application.

---

## Features

### 1. Account Creation Screen
- Create a new bank account
- Fields: Account Number, Holder Name, Initial Balance, KYC Status
- Validates input before account creation

### 2. Transaction Screens
- Deposit money
- Withdraw money
- Transfer money to another account
- Transfer is allowed only if:
  - Sender KYC is verified
  - Sufficient balance is available

### 3. Account Listing Screen
- Displays all created accounts
- Shows account number, holder name, balance, and KYC status

### 4. Output Display Panel
- Displays success and error messages
- Shows system responses clearly for every action
